JSML - Javascript Markup Language

Javascript has been a scripting language for so long, but now we are upgrading it.
We have added new functionalities and syntax to Javascript, And made JSML.
Here is an example of JSML code: 

<div JSML>
    p(Welcome to JSML!);
    <p>JSML also supports normal HTML inside!</p>
    p(JSML has a similiar syntax to JSX, and easily allows you to paint(blue, color) text!);
</div>
The output would be: 
<div JSML compiled>
    <p>Welcome to JSML!</p>
    <p>JSML also supports normal HTML inside!</p>
    <p>JSML has a similiar syntax to JSX, and easily allows you to color text!</p>
</div>
This is a live JSML example of the code shown above:
