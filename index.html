JSML - Javascript Markup Language

Javascript has been a scripting language for so long, but now we are upgrading it.
We have added new functionalities and syntax to Javascript, And made JSML.
Here is an example of JSML code: 


            

                p(Welcome to JSML!);
                
JSML also supports normal HTML inside!

                p(JSML has a similiar syntax to JSX, and easily allows you to blue(color) text!);
The output would be: 
            
                

                    
Welcome to JSML!

                    
JSML also supports normal HTML inside!

                    
JSML has a similiar syntax to JSX, and easily allows you to color text!
